# Software Requirement Specification (SRS)

# Smart PG & Mess Management System

## Functional Requirements

### Authentication
- Owner Login
- Student Login
- Secure Logout

### PG Configuration
- Configure meal prices
- Configure deduction policy
- Configure billing date
- Configure payment methods

### Student Management
- Add Student
- Edit Student
- Remove Student
- Activate/Deactivate Student

### Meal Plan Management
- Breakfast
- Lunch
- Dinner
- Any meal combination

### Attendance
- Student marks future absence
- Owner views attendance
- Attendance history

### Billing
- Automatic monthly bill generation
- Meal-wise deduction
- Absence deduction
- Bill history

### Payments
- Record payments
- Upload payment screenshot
- OCR verification
- Payment approval
- Pending payment tracking

### Complaints
- Raise complaint
- Resolve complaint
- Complaint history

### Inventory
- Add inventory
- Update stock
- Low stock alerts

### AI Features
- Food demand prediction
- Inventory prediction
- AI Chat Assistant

### Reports
- Revenue
- Attendance
- Payments
- Food consumption

## Non-Functional Requirements

- Secure Authentication
- Fast Response Time
- Easy to Use
- Responsive UI
- Scalable Architecture
- Reliable Database
- Modular Code
- Backup Support

## User Roles

### Owner

- Configure PG
- Manage Students
- Manage Payments
- Generate Bills
- Handle Complaints
- View Reports

### Student

- Login
- View Bill
- Mark Absence
- Upload Payment
- Raise Complaint
- Chat with AI
